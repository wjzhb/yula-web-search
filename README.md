# 🔍 yula-web-search

<img src="./img/logo.png" width="100" alt="yula-web-search logo">
===========
No API key required web search skill for OpenClaw.

Features
--------

- ❌ **No API key required** — works out of the box
- 🔍 **Bing search as primary, Google as fallback**
- 📝 **Automatic content extraction** from top relevant results
- 🧠 **Summarize into complete answer** — no need to open links
- 🇨🇳 **Optimized for Chinese language queries**
- 📦 Works with local network/proxy — no external API required
